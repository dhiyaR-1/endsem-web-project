# 💊 Pharmacy Management Website

A multi page pharmacy management website developed as part of a team for our end semester User Interface Design (UID) course.

The project was designed to provide a simple and user friendly platform where users can register, browse medicines, manage their cart, view orders, and contact customer support. Working on this project gave us practical experience in building a complete front end web application and collaborating as a team.

# Features

- User registration with basic form validation
- Medicine catalogue
- Add to Cart functionality
- Checkout page
- Order summary
- Contact & Support page
- Multi-page website navigation

# Technologies Used

- HTML5
- CSS3
- JavaScript
